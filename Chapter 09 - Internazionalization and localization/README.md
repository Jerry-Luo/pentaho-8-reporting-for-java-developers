Internationalization and localization
===

This folder contains the Pentaho reports developed in Chapter 9 about internationalization and localization. To edit (and preview) the reports use the Pentaho Report Designer or any other Pentaho Reporting tool able to manage `prpt` files.

# Disclaimer

All the examples contained into this repository are developed using an Ubuntu Operating System v16.04 LTS with 4Gb of RAM and Processor Intel i7. 

The environment is composed by: Java JVM 1.8.0_131, Apache Maven 3.3.9, git version 2.7.4.
